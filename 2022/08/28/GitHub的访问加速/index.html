GitHub的访问加速

  • GitHub被墙的原理：

    github不同于YouTube等，并未完全被墙，而是通过dns污染让用户计算机无法通过域名获得网站的真实ip，而无法访问。IP的获取有两种方式，电脑本地的host（数量少，速度快），和dns服务器（速度慢，维护方便，可解析大量网址）。一般用户电脑的host文件并没有写入太多ip，而dns服务器的对相关网站的解析被污染，所以访问慢或者无法访问。

  • 通过将GitHub的ip写入本地电脑host文件，使用户电脑准确访问网站。
