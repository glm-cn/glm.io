    • 安装插件hexo-asset-image
    • 打开_config.yml文件，修改下述内容
    • 示例代码格式：
    • 但是依旧存在问题：hexo-asset-image插件长时间未更新，要做以下修改：
      • 进入你博客的根目录，然后下面顺序找到 index.js:

        node_modules –> hexo-asset-image –> index.js

      • 用VS Code 或者 记事本打开 index.js,在第 58 行，可以找到这么一行代码：
        $(this).attr('src', config.root + link + src);
      • 把这一行代码改成下面这样
        $(this).attr('src', src);
      • 保存文件
      • 后续插入图片格式，图片存放在相同目录的同名文件夹下
